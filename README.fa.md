# سیستم مدیریت مسابقات

این یک سیستم مدیریت مسابقات مبتنی بر جنگو است که به کاربران امکان ایجاد و مدیریت مسابقات، شرکت در مسابقات و تعامل با سایر کاربران از طریق یک سیستم چت داخلی را می دهد. این پروژه همچنین شامل یک سیستم کیف پول برای مدیریت هزینه های ورودی مسابقات و جوایز است.

## امکانات

*   **مدیریت مسابقات:** ایجاد، ویرایش و حذف مسابقات. پشتیبانی از مسابقات انفرادی و تیمی.
*   **مدیریت مسابقات:** تولید خودکار مسابقات، تأیید نتایج و حل اختلافات.
*   **مدیریت کاربران:** ثبت نام کاربر، احراز هویت و مدیریت پروفایل. کنترل دسترسی مبتنی بر نقش (مدیر، کاربر).
*   **مدیریت تیم:** ایجاد و مدیریت تیم ها، افزودن یا حذف اعضا.
*   **سیستم کیف پول:** مدیریت کیف پول کاربران، شامل واریز، برداشت، هزینه های ورودی و جوایز.
*   **سیستم چت:** چت زنده بین کاربران.
*   **سیستم اطلاع رسانی:** اطلاعیه های زنده برای کاربران.
*   **مستندات API:** مستندات API به طور خودکار با استفاده از `drf-spectacular` تولید می شود.

## شروع به کار

### پیش نیازها

*   پایتون 3.8+
*   PostgreSQL
*   Redis

### نصب

1.  **کلون کردن مخزن:**

    ```bash
    git clone https://github.com/your-username/tournament-project.git
    cd tournament-project
    ```

2.  **ایجاد و فعال سازی یک محیط مجازی:**

    ```bash
    python -m venv venv
    source venv/bin/activate
    ```

3.  **نصب وابستگی ها:**

    ```bash
    pip install -r requirements.txt
    ```

4.  **تنظیم متغیرهای محیطی:**

    یک فایل `.env` در دایرکتوری `tournament_project` ایجاد کنید و متغیرهای زیر را اضافه کنید:

    ```
    SECRET_KEY=your-secret-key
    DEBUG=True
    DATABASE_URL=postgres://user:password@host:port/dbname
    REDIS_URL=redis://localhost:6379/0
    ZARINPAL_MERCHANT_ID="your-merchant-id"
    ZARINPAL_SANDBOX=True
    ```

5.  **اجرای مایگریشن های پایگاه داده:**

    ```bash
    python manage.py migrate
    ```

6.  **ایجاد یک کاربر superuser:**

    ```bash
    python manage.py createsuperuser
    ```

7.  **اجرای سرور توسعه:**

    ```bash
    python manage.py runserver
    ```

API در آدرس `http://127.0.0.1:8000/api/` در دسترس خواهد بود.

### مستندات API

مستندات API در آدرس `http://127.0.0.1:8000/api/docs/` در دسترس است.

## ساختار پروژه

این پروژه به چندین برنامه جنگو تقسیم شده است:

*   `chat`: چت زنده بین کاربران را مدیریت می کند.
*   `notifications`: اطلاعیه های کاربران را مدیریت می کند.
*   `tournaments`: برنامه اصلی برای مدیریت مسابقات و مسابقات.
*   `users`: کاربران، تیم ها و احراز هویت را مدیریت می کند.
*   `wallet`: کیف پول و تراکنش های کاربران را مدیریت می کند.
